Wat is HTML

Bij het maken van Deze website Hebben wij gebruik gemaakt van Html.
HTML is een afkorting voor HyperText Markup Language , het is gemaakt om texten op het world wide web te kunnen plaatsen.
Om de opmaak van zo'n text te verbeteren is CSS ontworpen.

opmaak

Daarnaast is HTML een opmaaktaal zoals vele andere, met notaties voor het aangeven van tekst, van kopjes, van indeling in paragrafen, van tabellen en van afbeeldingen en multimedia (die echter zelf niet in HTML worden gespecificeerd). HTML bestaat uit platte tekst waarin met markeringstekens is aangegeven hoe de tekst moet worden geschreven, bijvoorbeeld als lijst of als opschrift. Zo'n markering wordt een tag genoemd. HTML wordt meestal bekeken met een webbrowser, een programma dat HTML-documenten opvraagt en als opgemaakte tekst aan de gebruiker toont. In de loop der jaren is het aantal verschillende tags dat in HTML wordt gebruikt, enorm uitgebreid. De oorspronkelijke aanbeveling is een aantal malen geactualiseerd in verband met verdere ontwikkeling van HTML. De laatst geaccepteerde aanbeveling, HTML 5.0, is van oktober 2014.
